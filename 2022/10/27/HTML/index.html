HTML

简介：

  • html(Hypertext Markup Language):超文本标记语言

  • HTML不是一种编程语言，而是一种标记语言（markup language)

  • 标记语言是一套标记标签（markup tag）

  • HTML使用标记标签来描述网页

1.html的基本骨架

  • html:整个网页
  • head：网页头部，用来存放给浏览器看的信息，如css
  • body:网页主体，用来存放给用户看的信息，如图片、文字。

2.html元素

元素：开始标签+内容+结束标签

  • 标签成对出现
  • <>里放标签的英文
  • 结束标签比开始标签多一个/
  • 拓展：
    • 空元素：不在开始标签和结束标签之间放置任何内容
    • 双标签（嵌套的html元素）：成对出现的标签
    • 单标签（空元素）：只有开始标签，没有结束标签，如：<br>,<hr>

3.HTML属性

元素可以通过其属性进行配置

a元素创建一个超链接，被点击加载不同的HTML文档

属性引用

  • 要为包含引号的属性指定值
  • 属性有一个名称和一个值
  • 属性只能添加到起始标签或单个标签
  • 属性永远不可能添加到结束标签
  • 有一组可应用于任何html元素的全局属性
  • 元素可以定义自己的属性，提供特定于元素的配置信息

具有多个属性的元素

可以将多个属性应用于元素通过一个或多个空格分隔它们

  • 属性的顺序不重要
  • 上述代码，class和id是全局属性，可以将全局属性和特定属性混合使用

布尔属性

布尔属性是没有值的属性

disabled是一个布尔属性

可以通过分配空字符串(“”)来定义相同的布尔属性或将该值设置为属性的名称。

4.标签

1.标题标签：h1 h2….h6

2.段落标签：<p></p>

  • 独占一行
  • 段落之间存在一个空行

3.换行和水平线标签

  • 换行：<br>或<br />

  • 水平线：<hr>

4.文本格式化标签：

作用：为文本添加特殊格式，以突出内容，常见的文本格式：加粗，倾斜，下划线，删除线等。

  • 多使用第一个表格中的标签，语义更强

5.图像标签

作用：在网页中插入图片

  • src:源属性，指定图片的url
  • alt:替代性文本属性，在图片无法加载时会显示替代性文本
  • title:鼠标悬停会显示该文本
  • width height:调整尺寸
  • algin:设置对齐方式

相对路径：

写法 含义
. 当前文件所在文件夹
.. 当前文件上一级文件夹
/ 进入某一个文件夹

绝对路径：

  • windows从盘符出发
  • Mac从根目录（/）出发

6.超链接：

作用：点击跳转到其他页面（本地文件，网络地址）

  • href属性:（Hypertext Reference）超文本引用

  • target属性：可以定义被链接的文档在何处显示（在新页面打开，还是原页面打开）

    值 作用
    _blank 新页面
    _parent 上一级页面
    _self 自身
    _top 第一层框架
    framename
  • id属性：可用于在一个HTML文档中创建书签标记

空链接：href=”#”

controls 显示音频控制面板
loop 循环播放
autoplay 自动播放 浏览器一般禁用自动播放功能

7.视频标签：

属性 作用 说明
src(必须属性) 视频url 支持格式：MP4，WebM,Ogg

muted 静音播放

8.列表

作用：布局内容排列整齐的区域

分类：无序，有序，自定义

  • 无序列表
    • 作用：布局排列整齐的不需要规定顺序的区域
    • 标签：ul嵌套li,ul是无序列表，li是列表条目
    • 注意：ul里面只能包裹li,li可以包裹任何内容
  • 有序列表
    • 作用：布局排列整齐的需要规定顺序的区域
    • 标签：ol嵌套li
  • 自定义列表
    • 标签：dl嵌套dt和dd,dl是定义列表，dt是定义列表的标题，dd是定义列表的描述/详情
    • dl里面只能包含dt和dd，dt、dd可以包含其他内容

tr 行

tips:在网页中，表格默认没有边框线，使用border属性可以为表格添加边框线

  • 表格结构标签：
    • 作用：用表格结构标签把内容划分区域，让表格结构更清晰，语义更清晰
标签名 含义 说明
thead 表格头部 表格头部内容
tbody 表格主体 表格主要区域
tfoot 表格底部 汇总信息区域
  • 合并单元格
    • 作用：将多个单元格合并成一个单元格，以合并同类信息

CSS

css定义：层叠样式表（Cascading Style Sheets）,是一样式表语言，用来描述HTML文档的呈现（美化内容）

css引入：

1.内部样式表：

直接在HTML书写位置：title标签下方添加style双标签，style标签里面书写css代码,即在head里添加。

2.外部样式表：

  • css代码写在单独的css文件中（.css）

  • 在HTML里使用link标签引入

3.行内样式:配合Javascript使用

  • css代码写在标签的style属性值里

1.基础选择器

  • 标签选择器

标签选择器：使用标签名作为选择器，选中同名标签设置相同的样式

  • 类选择器

作用：查找标签，差异化设置标签的显示效果

定义类选择器：.类名(注意点开头)

使用类选择器：标签添加 class="类名"

注意：一个标签只有一个class属性，但可以有多个值，多个值之间用空格隔开

  • id选择器

作用：查找标签，差异化设置标签的显示效果

场景：id选择器一般配合JavaScript使用，很少用来设置css样式

定义id选择器：#id名

使用id选择器：标签添加id="id名"

  • 通配符选择器

作用：查找页面所用标签，设置相同的样式

通配符选择器：*，不需要调用，浏览器自动查找页面所有标签，设置相同的样式

3.复合选择器

定义：由两个或者多个选择器组成的，通过不同的方式组合

作用：更准确、更高效的选择目标元素（标签）

1**.后代选择器**

后代选择器：选中某元素的后代元素（所有后代）

写法：父选择器 子选择器 {css属性},父子选择器之间用空格隔开

2.子代选择器

子代选择器：选中某元素的子代元素（最近的子代）

写法：父选择器 > 子选择器{css属性},父子选择器之间用>隔开

3.并集选择器

并集选择器：选中多组标签设置相同的样式。

选择器写法： 选择器1，选择器2，..... ,选择器N{css属性},选择器之间用,隔开。

4.交集选择器

交集选择器：选中同时满足多个条件的元素

选择器写法：选择器1选择器2{css属性},选择器之间连写，没有任何符号

注意：如果交集选择器中有标签选择器，标签选择器必须书写在最前面。

4.伪类选择器

伪类选择器：伪类表示元素状态，选中元素的某个状态设置样式

鼠标悬停状态：选择器：hover{css属性}

拓展：伪类-超链接

选择器 作用
：link 访问前
：visited 访问后
：hover 鼠标悬停
：active 点击时（激活）

提示：如果要给超链接设置以上四个状态，需要按LVHA的顺序书写

5.css特性

1.继承性：子级标签默认继承父级标签的文字控制属性，子级标签已有属性不会进行继承。

2.层叠性：

特点：

  • 相同的属性会覆盖：后面的css属性会覆盖前面的css属性
  • 不同的属性会叠加：不同的css属性都生效

3.优先级

  • 优先级：也叫权重，当一个标签使用了多种选择器时，基于不同种类的选择器的匹配规则。

  • 公式：通配符选择器 < 标签选择器 < 类选择器 < id选择器 < 行内样式 < ！important（选中标签的范围越大，优先级越低）

  • 优先级-叠加运算

    如果选择器是复合选择器，则需要权重叠加计算

    公式：（行内样式，id选择器个数，类选择器个数，标签选择器个数）

    规则：

    • 从左往右依次比较个数，同一级个数多的优先级高，如果个数相同，则向后比较
    • ！important优先级最高
    • 继承权重最低

7.显示模式

显示模式：标签（元素）的显示方式

作用：布局网页的时候，根据标签的显示模式选择合适的标签摆放位置

  • 块级元素

    • 独占一行
    • 宽度默认是父级的100%
    • 添加宽高属性生效

  • 行内元素

    • 一行可以共存多个
    • 宽高属性不生效
    • 宽高由内容撑开

  • 行内块元素

    • 一行可以共存多个
    • 宽高属性生效
    • 宽高默认由内容撑开

转换显示模式：

属性名：display

属性值：

属性名 效果
block 块级
inline-block 行内块
inline 行内
